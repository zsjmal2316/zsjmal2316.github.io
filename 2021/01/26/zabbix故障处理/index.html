zabbix故障处理

故障1：zabbix server is not running

故障原因:

zabbix-server的配置文件里配有数据库密码，需要配置对密码

查看报错的日志

故障解决：

添加正确的数据库账号密码信息

故障2：no module named requests

故障现象：微信报警失败

故障原因

缺少模块 requests

故障解决

安装缺失的依赖包

故障3：zabbix_get测试提示警告：No route to host

故障现象：在server端使用zabbix_get命令测试键值命令时提示警告

故障原因

主机开启了防火墙或者是SELinux

故障解决

关闭防火墙和SELinux

故障4：zabbix_get测试提示警告：Not all processes could be identified

故障现象：在server端使用zabbix_get命令测试键值命令时提示警告

故障原因

zabbix_agent是以普通用户zabbix运行的，而普通用户执行netstat -antp时会有警告，网上查找发现只要不是用p参数就可以普通用户运行

故障解决

监控脚本里的命令修改为netstat -ant
